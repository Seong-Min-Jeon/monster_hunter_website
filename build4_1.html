교체 기술2: 귀인화[짐승]

교체 기술3: 벌레철사 베기

주력 기술 커맨드
1. 귀인공무: 귀인화, 귀인강화 상태에서 A (방향키 상관없음)
2. 귀인공선연참: 귀인공무 히트 시 자동 발생
3. 공중 회전베기: 귀인화 상태, 공중에서 X
4. 공중 회전난무: 귀인화, 귀인강화 상태, 공중에서 A
5. 회전난무 피니시: 귀인상태의 공중 회전난무 중 X or A
6. 점프2단베기: 귀인강화 상태, 공중에서 X
7. 귀인화[짐승]: ZR
8. 귀인난무: 귀인화 상태에서 X+A
8. 귀인연참: 귀인강화 상태에서 X+A(X+A X+A)
콤보 루트
1. (귀인화 상태) 귀인공무 -> 귀인공선연참 -> 공중 회전베기 -> 공중 회전난무
2. (귀인강화 상태) 귀인공무 -> 귀인공선연참 -> 공중 회전난무 -> 회전난무 피니시
3. (귀인강화 상태) 귀인공무 -> 귀인공선연참 -> 점프2단베기
4. (귀인강화 상태) 귀인공무 -> 귀인공선연참 -> 귀인화[짐승] -> 공중 회전난무
운영법
1. 귀인강화를 항상 발동시킨다.
2. 회피와 귀인화를 조절하면서 스테미나 관리를 해야한다. (스테미나가 0이면 회피를 사용할 수 없다.)
3. 귀인공무를 헛치지 않는 것이 중요하다. -> 헛친 상태에서 A를 연타하면 추가타가 나가기 때문에 빈틈이 생길 수도 있다.
4. 맞고 날라가는 중에 밧줄벌레 낙법을 사용, 착지 전 귀인화를 사용한다. 5. 국소적인 약점 부위를 공격할 타이밍이 오면 귀인난무, 귀인연격을 사용한다.
솔플 운영법
1. 몬스터가 후딜이 있는 기술을 헛치게 유도하고 콤보를 사용한다.
2. 밧줄벌레 낙법을 몬스터 방향으로 사용할 시 맞을 가능성이 높다. (나를 인식하고 있을 가능성이 높기 때문)
3. 숫돌은 몬스터가 도망갔을 때 주로 사용하며, 급할 때에는 콤보를 한번 포기하고 사용한다.
파티 운영법
1. 나를 인식하고 있지 않다면 몬스터의 꼬리/머리쪽으로 이동해 콤보를 사용한다.
2. 콤보 후에는 대부분 플레이어가 몬스터의 머리/꼬리 부분에 위치한다.
2-1. 2번 상황에서 플레이어가 꼬리 부분에 위치한다면 콤보를 한번 더 써준다.
2-2. 2번 상황에서 플레이어가 머리 부분에 위치한다면 몬스터의 행동에 따라 선택지가 나뉜다.
2-2-1. 돌진형 기술이 아니며 보고 반응할 수 있는 수준의 공격을 쓴다면 귀인공무의 무적 프레임으로 씹고 콤보를 쓴다.
2-2-2. 돌진형 기술 등 귀인공무 히트 이후 콤보가 불가능한 상황일 경우 거리를 벌리고 딜캐로 귀인공무를 쓴다. (대부분 꼬리쪽에서 사용하게 된다.)
2-2-3. 공격을 할 수 있는 상황이 아니라면 거리를 벌리고 다시 기회를 잡는다.
